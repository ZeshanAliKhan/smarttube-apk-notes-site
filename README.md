# SmartTube APK Notes

SmartTube APK Notes is a GitHub Pages-ready static site scaffold for a focused `smarttube apk` content project.

This stage only sets up the project structure, page map, publish files, and internal linking plan. The full content draft comes next after approval.

## What is included

- static HTML, CSS, and JavaScript only
- one homepage
- three SmartTube-focused content pages
- GitHub Pages files
- docs for publishing, site structure, and image planning
