# Publish Guide

## Expected repository name

`smarttube-apk-notes-site`

## Expected GitHub Pages URL

`https://zeshanalikhan.github.io/smarttube-apk-notes-site/`

## Steps

1. Create a public GitHub repository with the same name.
2. Push the local project folder to `main`.
3. Open repository `Settings`.
4. Open `Pages`.
5. Set:
   - `Deploy from a branch`
   - `main`
   - `/root`
6. Save.
7. Wait for the live Pages URL to return `200`.

## If the repository name changes

Update these files:

- `index.html`
- `pages/*.html`
- `robots.txt`
- `sitemap.xml`
- `docs/site-map.md`
